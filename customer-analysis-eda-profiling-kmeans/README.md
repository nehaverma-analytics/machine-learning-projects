# Customer Segmentation using EDA, Data Profiling and K-Means Clustering

## Project Overview

This project performs exploratory data analysis (EDA), automated data profiling, and customer segmentation using the K-Means clustering algorithm. The goal is to identify meaningful customer groups based on behavioral and demographic patterns to support data-driven marketing and personalization strategies.

The notebook demonstrates a complete workflow from data exploration to unsupervised learning for segmentation.

---

## Objectives

- Perform detailed exploratory data analysis (EDA)
- Generate automated dataset profiling
- Identify patterns in customer behavior
- Segment customers using K-Means clustering
- Interpret clusters for business insights

---

## Workflow
